# AutoCompress (Kettu / Revenge / Bunny)

Keeps Discord mobile video uploads under **20MB** (configurable). Oversized clips go through **ezgif.com**’s video compressor, then re-attach to Discord when possible.

## Install

```
https://nickb926.github.io/kettu-auto-compress/AutoCompress
```

Refetch after updates. Settings: target MB, enable/disable ezgif, videos/images toggles.

## Notes

Pure Kettu plugins can’t ship FFmpeg. ezgif is unofficial site automation and can take 1–3 minutes on large clips.
